# Paithon's world modpack

A simple building modpack i used for my building world

# Installation

- Go to releases page https://modrinth.com/modpack/paithons-world-modpack/releases

## PrismLauncher/Modrinth

- Download latest release pack file (.mrpack)
- Import the pack file into [PrismLauncher](https://prismlauncher.org) or [Modrinth app](https://modrinth.com/app)

## Cracked Launcher

* Download the executable installer file
- For Windows: use instailler.exe
- For Mac/Linux: use installer executable file
* Run the installer
* Navigate the game launcher to the modpack directory
